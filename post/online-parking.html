Online Parking

This is my computer engineering final year project. I started this project when I first learned .Net core. I was the sole creator/developer for the project.

The system is mainly focused on providing generic parking solution for entire city. Normal users can book parking slot from available parking lots in particular area as well as search near by parking lots. Parking owners add their parking lots, after approval parking lots will be shown on search results and user will be able to book slots in this newly added parking lots .

System has eye catching UI which gives user positive impresion with highly secure backend system to provide seemless flow of booking slot

Parking owners have access to view statistics for their parking lot and ability to manage parking lot like add/remove slot disable booking for perticular slot or interval etc...

I worked on this system for 1 years.
